วิธีการใช้งานไลน์บอท

ขั้นตอนที่ 1
เลือกเมนูในการใช้งานว่าจะเลือกการตรวจโรคเป็นโหมดใด โดยจะมีทั้งหมด 2 โหมด คือ
1.Fundus Mode(เป็นภาพถ่ายส่วนหลังของตาที่เกี่ยวข้องกับจอประสาทตา และ เส้นประสาทที่ออกจากส่วนหลังของดวงตา เพื่อตรวจสอบภาวะต่างๆ)
2.OCT Mode(เป็นภาพทางการแพทย์ที่ใช้แสงสะท้อนเพื่อส่องโครงสร้างภายในตา เพื่อวินิจฉัยและติดตามภาวะต่างๆ ในระบบตา)
ขั้นตอนที่ 2
หลังจากเลือกโหมดแล้ว แชทบอทจะตอบกลับเป็น"ชื่อโหมด Mode OK" ดังนี้
1.Fundus Mode OK
2.OCT Mode OK
ขั้นตอนที่ 3 เลือกและส่งรูปภาพ
เมื่อแชทบอทตอบกลับมา แสดงว่าพร้อมสำหรับการใช้งานแล้ว ให้กดเลือกรูปจอประสาทตาที่จะส่งแล้วกดส่งได้เลย โดยตัวอย่างที่
1. คือ ภาพจอประสาทตาของ fundus mode ที่เป็นปกติ
2. คือ ภาพจอประสาทตาของ fundus mode ที่เป็นโรค
3. คือ ภาพจอประสาทตาของ oct mode ที่เป็นปกติ
4. คือ ภาพขอประสาทตาของ oct mode ที่เป็นโรค
ขั้นตอนที่ 4 แสดงผลลัพธ์
เมื่อส่งรูปภาพไปแล้ว ตัวปัญญาประดิษฐ์จะใช้เวลาซักครู่ในการวินัจฉัยโรค แล้วจึงตอบกลับมาว่าเจอโรคหรือไม่พร้อมกับส่งรูปที่ทำการตรวจแล้วออกมา
โดยผลลัพธ์ที่เป็นไปได้จะมีทั้งหมด 4 รูปแบบคือ
1.fundus mode "normal found!" คือ โหมด fundus พบว่าจอประสาทตาปกติ
2.fundus mode "disease found!" คือ โหมด fundus พบว่าจอประสาทตาเป็นโรค
3.oct mode "normal found!" คือ โหมด oct พบว่าจอประสาทตาปกติ
4.oct mode "disease found!" คือ โหมด oct พบว่าจอประสาทตาเป็นโรค

ฟังก์ชั่นอื่นๆ

โหมดที่กำลังใช้งานอยู่
เมื่อกดที่คำสั่ง "โหมดที่กำลังใช้งานอยู่" ตัวแชทบอทก็จะตอบกลับมาว่าตอนนี้เป็นโหมดตรวจแบบไหนอยู่ โดยผลจะมีอยู่สองประเภท คือ Fundus Mode และ OCT Mode ดังภาพ
ผู้จัดทำ
เมื่อกดที่คำสั่ง "ผู้จัดทำ" ตัวแชทบอทจะพาผู้ใช้งานไปยังเว็บไซต์แสดงชื่อและภาพของผู้จัดทำและอาจารย์ที่ปรึกษา
